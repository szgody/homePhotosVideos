# homePhotosVideos
这是一个使用 Vue.js 构建的家庭照片与视频网站应用。该应用允许用户查看和管理家庭照片和视频。
home-photo-vue/
├── backend/
│   ├── src/
│   │   ├── routes/
│   │   ├── services/
│   │   └── utils/
│   ├── package.json
│   └── server.js
└── public/
    ├── images/
    │   ├── original/    # 原始图片
    │   └── processed/   # 处理后的图片
    └── videos/
        ├── original/    # 原始视频
        └── processed/   # 处理后的视频
